# OfficeLensOnMatlab
Office Lens On Matlab

- 本程序类似Microsoft Office Lens，
- 用于在缺乏专业扫描仪的情况下，自动分析截取照片并且，提供图像修正和OCR的功能。 Copyright@ JW. Zhang
